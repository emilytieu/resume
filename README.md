# Emily Tieu's Personal Website
https://emilytieu.github.io/resume/

A simple personal website to showcase my background, skills, projects, and professional experience. This site serves as an online resume and portfolio, making it easy for visitors, recruiters, and potential collaborators to learn more about me.

## Features
- Professional introduction and bio
- Skills and technologies overview
- Links to my CV, GitHub, and LinkedIn
- Overview of past and current experiences
- Portfolio of projects
- List of received awards
- Toggle view of courses I've taken and papers I've contributed to
- Hobbies I enjoy to maintain my work-life balance 
- Contact information
- Responsive design for desktop and mobile devices

Deployed via GitHub Pages.
